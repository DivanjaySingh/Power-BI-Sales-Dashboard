# Power-BI-Sales-Dashboard
Power BI Sales Dashboard project analyzing online sales performance across India. Includes KPI cards, state-wise profit analysis, monthly trends, and payment insights using DAX and interactive visuals.
Power BI Sales Dashboard

# Project Overview
This Power BI dashboard analyzes online sales performance across India.

 # Objective
To track sales, profit, and customer insights to support data-driven business decisions.

# Key Features
- KPI Cards (Total Sales, Profit, Quantity)
- State-wise Profit Analysis
- Monthly Profit Trends
- Category & Sub-Category Insights
- Payment Mode Analysis
- Interactive Filters & Slicers

# Tools Used
- Power BI
- DAX
- Excel


 # Learning Note
This project was built for learning purposes by following a tutorial and practicing Power BI skills.
